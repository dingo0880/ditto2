디토 시니어 친화형 키오스크 전체 패키지

구성:
- index.html : 첫 선택 화면
- hospital.html : 디토 병원 키오스크
- cafe.html : 디토 카페 키오스크
- README.txt

GitHub Pages 업로드 방법:
1. 이 ZIP을 압축 해제합니다.
2. index.html, hospital.html, cafe.html, README.txt 파일을 GitHub 저장소 루트에 업로드합니다.
3. Settings > Pages에서 main / root로 배포합니다.

주의:
- ZIP 파일 자체를 올리지 말고, 압축을 푼 파일들을 각각 올려야 합니다.
- cafe.html과 hospital.html은 정상 작동하던 최종본입니다.
- index.html은 병원/카페와 같은 태블릿 프레임 방식으로 맞춘 선택 화면입니다.
